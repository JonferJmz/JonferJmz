## 👨‍💻 Perfil Profesional

Soy Ingeniero en Sistemas con experiencia práctica en desarrollo de software, administración de servidores y automatización de procesos industriales. Mi trabajo abarca desde la configuración de infraestructura física y servidores locales, hasta el desarrollo de aplicaciones web y móviles. Me enfoco en resolver problemas técnicos de manera directa, creando herramientas funcionales y manteniendo sistemas operativos estables.

---

## 🏭 Experiencia y Proyectos Destacados

* **Automatización en Foxconn (POD2):** Administración de entornos **CentOS** y desarrollo de scripts en **Bash** para la gestión de racks, asignación de IPs y control de testers.
* **Herramientas de Monitoreo:** Desarrollo y mantenimiento de una WebApp operativa 24/7 (HTML/JS) sobre Git Bash, diseñada para el monitoreo y reinicio remoto de testers en piso.
* **Accesibilidad y Enrutamiento:** Creación de scripts ejecutables (`.bat`) para dirigir automáticamente a los usuarios operativos hacia las plataformas internas de monitoreo mediante IPs locales.
* **Infraestructura Propia:** Configuración y administración de un *home server* personal y gestión del dominio **jonfertech.com**.
* **Desarrollo Integral:** Contenerización de servicios con **Docker**, integración de IA local para proyectos mediante **Ollama**, y desarrollo de la aplicación móvil **MotoRiders** utilizando **Flutter**.
