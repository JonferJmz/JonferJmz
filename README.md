### 🛠️ Arquitectura & Desarrollo

Convierto código en arquitecturas escalables. Me especializo en el desarrollo backend, la automatización extrema y la integración perfecta entre hardware y software. 

* ⚙️ **Core:** Automatización profunda con **Bash**, desarrollo robusto en **Python** y despliegues seguros utilizando contenedores **Docker**.
* 🧠 **Innovación:** Implementación y despliegue de modelos de IA locales mediante **Ollama** para potenciar aplicaciones inteligentes.
* 📱 **Ecosistema:** Creación de APIs RESTful, gestión de bases de datos relacionales (SQL) y desarrollo de aplicaciones móviles multiplataforma con **Flutter**.
